AI Movie Recommendation System (NLP + Streamlit)  Built a content-based recommendation system using NLP techniques Used CountVectorizer and cosine similarity for similarity computation Developed an interactive web application using Streamlit Optimized performance using caching and efficient filtering

# 🎬 Movie Recommendation System

## 📌 Overview
This project recommends movies based on content similarity using NLP techniques.

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## 🚀 Features
- Search movies
- Get top 5 recommendations
- Fast and interactive UI

## 🧠 Algorithm
- CountVectorizer (NLP)
- Cosine Similarity
